# 보고서 및 분석 (Reporting & Analytics)

**보고서 및 분석 (Reporting & Analytics)** 기능은 프로젝트의 진행 상황, 성과, 리소스 활용도 등을 시각적으로 파악하고 데이터에 기반한 의사결정을 지원하는 기능입니다.

## 상세 기능 분석

### I. 대시보드 (Dashboard)

프로젝트의 핵심 지표(KPI)를 한눈에 파악할 수 있는 종합 대시보드를 제공합니다.

| 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- |
| **종합 현황** | **프로젝트 오버뷰** | 전체 프로젝트의 진행률, 마감 임박 작업, 지연된 작업 수 등을 요약하여 표시합니다. |
| **시각화** | **차트 및 그래프** | 파이 차트(상태별 작업), 바 차트(담당자별 작업 수) 등 다양한 차트 라이브러리(예: Recharts, Chart.js)를 활용하여 데이터를 시각화합니다. |
| **개인화** | **내 작업 요약** | 로그인한 사용자가 담당하고 있는 작업의 현황과 오늘 해야 할 일(Today's Tasks)을 우선적으로 표시합니다. |

### II. 진척도 및 성과 분석

프로젝트가 계획대로 진행되고 있는지 심층적으로 분석합니다.

| 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- |
| **진척도** | **번다운 차트 (Burndown Chart)** | 시간 경과에 따른 남은 작업량의 추이를 시각화하여 프로젝트 완료 가능 시점을 예측합니다. |
| **S-Curve** | **누적 진척 곡선** | 계획된 진척도(Planned Value)와 실제 진척도(Earned Value)를 비교하는 S-Curve를 제공합니다. |
| **EVM** | **획득 가치 관리** | 자원 및 비용 관리 섹션과 연동하여 CPI(비용 성과 지수), SPI(일정 성과 지수) 등 EVM 지표를 상세 보고서로 제공합니다. |

### III. 리소스 및 비용 보고서

자원 활용과 예산 집행 현황을 분석합니다.

| 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- |
| **리소스** | **리소스 활용도 맵** | 팀원별 업무 부하량을 히트맵(Heatmap)이나 바 차트로 시각화하여 과부하(Overallocation)된 자원을 식별합니다. |
| **비용** | **예산 집행 현황** | 항목별 예산 대비 실제 지출 비용을 비교 분석하고, 초과 지출이 예상되는 항목을 경고합니다. |

### IV. 내보내기 및 공유

분석 결과를 외부와 공유할 수 있는 기능을 제공합니다.

| 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- |
| **내보내기** | **PDF/Excel 내보내기** | 보고서 화면을 PDF로 저장하거나, 원본 데이터를 Excel(CSV) 형식으로 다운로드하여 2차 가공할 수 있도록 지원합니다. |
| **공유** | **보고서 링크 공유** | 읽기 전용(Read-only) 보고서 페이지 링크를 생성하여 이해관계자와 현황을 공유할 수 있습니다. |
