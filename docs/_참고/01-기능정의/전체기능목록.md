## 1. 인증, 사용자 관리 및 접근 제어 (Authentication, User Management & Security)
> 상세 분석 문서: [01-auth/인증 및 보안 전략.md](./01-auth/인증%20및%20보안%20전략.md)

ZeroPM은 **Supabase Auth**와 **RLS (Row Level Security)**를 기반으로 안전하고 비용 효율적인 인증 체계를 구축합니다.

| 기능 ID | 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- | :--- |
| AUTH-01 | **인증** | **이메일/비밀번호 로그인** | Supabase Auth를 활용한 기본 계정 생성 및 로그인. |
| AUTH-02 | | **소셜 OAuth 로그인** | Google, GitHub 등 소셜 공급자 연동 지원. |
| AUTH-03 | **접근 제어** | **RLS (Row Level Security)** | PostgreSQL RLS를 통해 인증된 사용자별 데이터 접근 권한을 DB 레벨에서 강제. |
| AUTH-04 | | **RBAC (역할 기반 접근 제어)** | `admin`, `member` 등 역할에 따른 세밀한 권한 제어 정책 적용. |
| AUTH-05 | **사용자 관리** | **프로젝트 멤버 초대** | 이메일 기반 멤버 초대 및 역할 할당 기능. |
| AUTH-06 | | **무료 티어 활용** | Supabase Free Tier (월 50,000 MAU) 범위 내 운영 최적화. |

## 2. 실시간 협업 및 데이터 무결성 (Collaboration & Integrity)
> 상세 분석 문서: [02-collaboration/실시간 협업 및 데이터 무결성.md](./02-collaboration/실시간%20협업%20및%20데이터%20무결성.md)

**CRDT (Yjs)**와 **Supabase Realtime**을 결합하여 충돌 없는 동시 편집 환경을 제공합니다.

| 기능 ID | 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- | :--- |
| COL-01 | **동시 편집** | **CRDT 기반 동기화** | Yjs + Supabase Realtime을 통한 데이터 자동 병합 및 충돌 방지. |
| COL-02 | **사용자 인식** | **프레즌스 (Presence)** | 현재 접속 중인 사용자와 편집 중인 Task를 실시간 시각화 (Soft Lock). |
| COL-03 | **UX** | **Optimistic UI** | 서버 응답 대기 없이 즉각적인 UI 업데이트로 체감 성능 극대화. |
| COL-04 | | **오프라인 지원** | IndexedDB/RxDB 활용 로컬 저장 및 네트워크 복구 시 자동 동기화. |
| COL-05 | **데이터 무결성** | **낙관적 잠금** | 구조적 변경(삭제 등) 시 `version` 필드를 통한 충돌 감지. |
| COL-06 | | **트랜잭션 처리** | 복합적인 데이터 변경을 PostgreSQL 트랜잭션(RPC)으로 원자적 처리. |

## 3. 핵심 프로젝트 관리 및 WBS (PM Core & WBS)
> 상세 분석 문서: [03-wbs/핵심 프로젝트 관리 및 WBS.md](./03-wbs/핵심%20프로젝트%20관리%20및%20WBS.md)

무한 계층 구조의 WBS와 기준선(Baseline) 관리를 통해 체계적인 프로젝트 계획을 지원합니다.

| 기능 ID | 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- | :--- |
| PM-01 | **WBS** | **무한 계층 구조** | 인접 리스트 모델 및 재귀 CTE를 활용한 깊이 제한 없는 Task 계층 지원. |
| PM-02 | | **자동 롤업 (Roll-up)** | 하위 Task의 일정, 비용, 진척률을 상위 요약 작업으로 자동 집계. |
| PM-03 | **Task 유형** | **Custom Task Type** | Milestone, Summary 외 사용자 정의 유형(`urgent` 등) 및 스타일링 지원. |
| PM-04 | **기준선** | **다중 기준선 (Baseline)** | 초기 계획 스냅샷 저장 (최대 11개) 및 현재 계획과의 편차 분석. |
| PM-05 | | **중간 계획 (Interim Plan)** | 진행 중 특정 시점의 일정 스냅샷 저장 (최대 10개). |
| PM-06 | **의존성** | **4가지 의존성 유형** | FS, SS, FF, SF 및 Lag/Lead Time 지원. |

## 4. 고급 스케줄링 및 엔진 (Advanced Scheduling Engine)
> 상세 분석 문서: [04-scheduling/고급 스케줄링 및 엔진.md](./04-scheduling/고급%20스케줄링%20및%20엔진.md)

클라이언트 측 고성능 엔진을 통해 CPM 계산 및 정밀한 일정 관리를 수행합니다.

| 기능 ID | 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- | :--- |
| SCH-01 | **그래프** | **DAG 모델링** | Graphlib 기반 Task/Dependency 그래프 구성 및 순환 참조(Cycle) 감지. |
| SCH-02 | **CPM** | **임계 경로 계산** | 전진/후진 계산을 통한 ES, EF, LS, LF, Total Float 산출 및 Critical Path 도출. |
| SCH-03 | **캘린더** | **업무일 기반 계산** | 주말 및 공휴일을 제외한 유효 작업 시간(Business Days) 기준 일정 산출. |
| SCH-04 | **성능** | **Web Worker 비동기 처리** | 복잡한 계산 로직을 백그라운드 스레드에서 실행하여 UI 블로킹 방지. |

## 5. 자원 및 비용 관리 (Resource & Cost Management)
> 상세 분석 문서: [05-resource/자원 및 비용 관리.md](./05-resource/자원%20및%20비용%20관리.md)

자원 충돌 감지 및 비용 성과 분석(EVM)을 통해 프로젝트 리소스를 최적화합니다.

| 기능 ID | 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- | :--- |
| RES-01 | **자원** | **자원 정의 및 할당** | 인력, 장비 등 갱신 가능 자원 정의 및 Task별 할당량(%) 설정. |
| RES-02 | **RCPSP** | **실시간 충돌 감지** | 자원 가용량 초과 시 실시간 경고 및 시각적 피드백 제공. |
| RES-03 | | **휴리스틱 조정 지원** | LFT, GRD 등 우선순위 기반 수동 조정 가이드 제공. |
| RES-04 | **비용** | **비용 데이터 관리** | 시간당 요율(Rate) 및 고정 비용(Fixed Cost) 입력 및 계산. |
| RES-05 | | **EVM 분석** | 비용 기준선 대비 편차(CV) 및 성과 지표(CPI) 분석. |

## 6. 사용자 인터페이스 및 아키텍처 (UI/Architecture)
> 상세 분석 문서: [06-ui/사용자 인터페이스.md](./06-ui/사용자%20인터페이스.md)

Next.js와 가상화 기술을 통해 대용량 데이터에서도 부드러운 사용자 경험을 제공합니다.

| 기능 ID | 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- | :--- |
| UI-01 | **Gantt** | **가상화 렌더링** | SVAR React Gantt 활용, 행/타임라인 가상화로 수천 개 Task 고성능 렌더링. |
| UI-02 | **뷰** | **다양한 뷰 모드** | Gantt 외 Kanban Board, Calendar View 제공. |
| UI-03 | **그리드** | **가상화 데이터 그리드** | 대용량 Task 목록의 부드러운 스크롤 및 편집 지원. |
| UI-04 | **아키텍처** | **분리형 아키텍처** | Logic(엔진)과 View(UI)의 명확한 분리, Serverless 백엔드(Supabase) 활용. |

## 7. 보고서 및 분석 (Reporting & Analytics)
> 상세 분석 문서: [07-reporting/보고서 및 분석.md](./07-reporting/보고서%20및%20분석.md)

프로젝트 현황을 한눈에 파악하고 데이터 기반 의사결정을 지원하는 대시보드 및 리포팅 기능을 제공합니다.

| 기능 ID | 기능 분류 | 상세 기능 | 설명 및 기술적 근거 |
| :--- | :--- | :--- | :--- |
| REP-01 | **대시보드** | **종합 현황판** | 프로젝트 진행률, 마감 임박 작업, 지연 작업 등 핵심 지표(KPI) 요약 표시. |
| REP-02 | | **개인화 위젯** | 사용자별 '내 작업', '오늘의 할 일' 등 개인화된 뷰 제공. |
| REP-03 | **진척도** | **번다운/번업 차트** | 시간 경과에 따른 작업 완료 추이 및 예상 완료일 예측 시각화. |
| REP-04 | | **S-Curve** | 계획 대비 실제 진척도를 누적 곡선으로 비교 분석. |
| REP-05 | **리소스** | **활용도 분석** | 팀원별 업무 부하량 및 가용성을 시각화하여 리소스 평준화 지원. |
| REP-06 | **공유** | **내보내기** | 보고서 및 차트를 PDF, Excel 등으로 내보내기 지원. |
