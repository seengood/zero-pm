# ZeroPM 로우파이 와이어프레임

> ASCII 기반 로우파이(lo-fi) 와이어프레임. 레이아웃·컴포넌트 구성·계층 구조 확정이 목적이며,
> 시각 디자인(색상·폰트·아이콘)은 이 문서의 범위 밖이다.
>
> **Figma 와이어프레임:** [ZeroPM 와이어프레임 (Figma)](https://www.figma.com/design/ScfkTRlYZL2Kz4K3IjTt9X/ZeroPM-%EC%99%80%EC%9D%B4%EC%96%B4%ED%94%84%EB%A0%88%EC%9E%84?m=auto&t=U9sxyHshLsfE8Y3w-6)
>
> 연관 문서: [05-화면흐름도.md](./05-화면흐름도.md) · [03-User-Flow-시나리오.md](../../01-유저스토리/03-User-Flow-시나리오.md)

## 표기 규칙

| 기호 | 의미 |
|------|------|
| `[ 텍스트 ]` | 버튼 |
| `[ 입력값           ]` | 텍스트 인풋 |
| `[✓] / [ ]` | 체크박스 (선택됨 / 미선택) |
| `(•) / ( )` | 라디오 버튼 (선택됨 / 미선택) |
| `▓▓▓▓░░░░` | 프로그레스 바 (채움 / 빔) |
| `████` | 간트 바 (현재 일정) |
| `▒▒▒▒` | 기준선 바 (반투명 오버레이) |
| `▼` | 드롭다운 |
| `[*]` | 강조 표시 (CP, 경고 등) |
| `┄┄┄` | 흐림 처리된 배경 |

## 21. 공통 상태 컴포넌트

### 21-1. 저장 상태 칩 (상단 우측)

```
저장 중...  [스피너]          ← 디바운스 500ms 후 전송 중
저장됨 ✓                      ← 성공 (2초 후 소멸)
저장 실패   [ 재시도 ] [ ✕ ]  ← 네트워크 오류
```

### 21-2. 토스트 알림 (우측 하단)

```
┌──────────────────────────────────────────────────┐
│  3개 후행 태스크가 자동으로 조정되었습니다.    [✕] │  ← 정보 (파란)
└──────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────┐
│  저장하지 못했습니다.          [ 재시도 ]  [✕]   │  ← 에러 (빨간)
└──────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────┐
│  기준선 '기준선 1'을 저장했어요.               [✕] │  ← 성공 (녹색)
└──────────────────────────────────────────────────┘
```

